# 社交消息模板

消息模板用于生成个性化的开场白和后续跟进消息。模板支持变量插入，可以根据对方 profile 自动填充。

## 模板格式

```yaml
templates:
  - name: "模板名称"
    text: "消息文本，支持 {{variable}} 变量"
    usage: "使用场景说明"
    variables:
      - name: "variable"
        description: "变量说明"
```

## 开场白模板

### 通用模板

```yaml
- name: "礼貌问候"
  text: |
    Привет! Меня зовут {{user_name}}. Я изучаю русский язык и ищу новых друзей. 
    Приятно познакомиться!
  usage: "首次接触，通用场景"
  variables:
    - name: "user_name"
      description: "用户名字"

- name: "直接友好"
  text: |
    Привет! Как дела? Я из Китая, учу русский язык. 
    Увидел(а) твои фотографии — очень красиво!
  usage: "对方照片质量高时"

- name: "兴趣开场"
  text: |
    Привет! Вижу, что ты любишь {{interest}}. Я тоже! 
    Ты из Москвы? Как тебе там жить?
  usage: "知道对方兴趣时"
  variables:
    - name: "interest"
      description: "共同兴趣"
```

### 针对有照片的用户

```yaml
- name: "赞美照片"
  text: |
    Привет! У тебя очень красивые фотографии! 
    Ты из Москвы? Я изучаю русский язык и хочу Practice с носителями.
  usage: "对方照片质量高时"

- name: "好奇提问"
  text: |
    Привет! Скажи, пожалуйста, ты из какого города? 
    Я изучаю русский и хочу лучше узнать Россию.
  usage: "想了解对方来自哪里时"
```

### 针对有共同兴趣的用户

```yaml
- name: "共同兴趣"
  text: |
    Привет! Вижу, что ты тоже любишь {{shared_interest}}. 
    Это здорово! Ты давно занимаешься этим?
  usage: "有共同兴趣时"
  variables:
    - name: "shared_interest"
      description: "共同兴趣"

- name: "音乐爱好者"
  text: |
    Привет! Ты любишь музыку — я тоже! 
    Какую музыку ты обычно слушаешь? 
    Можешь порекомендовать что-нибудь русское?
  usage: "对方喜欢音乐时"

- name: "旅行爱好者"
  text: |
    Привет! Вижу, что ты любишь путешествовать. 
    Я тоже! Ты был(а) в каких странах? 
    Какое место тебе больше всего понравилось?
  usage: "对方喜欢旅行时"
```

## 后续跟进模板

### 3天后跟进

```yaml
- name: "轻松问候"
  text: |
    Привет! Как дела? Не хочешь поболтать?
  usage: "3天未回复时的轻松跟进"
```

### 一周后跟进

```yaml
- name: "分享近况"
  text: |
    Привет! Как прошла неделя? Что нового?
    Я вот продолжаю учить русский — это непросто, но интересно!
  usage: "一周后跟进"

- name: "延续话题"
  text: |
    Привет! Помнишь, мы говорили о {{previous_topic}}? 
    Вот подумал(а) об этом снова. Как ты думаешь?
  usage: "延续之前对话话题"
  variables:
    - name: "previous_topic"
      description: "之前对话的话题"
```

```yaml
    Ты не хочешь встретиться и выпить кофе? 
    Было бы здорово познакомиться лично!

  usage: "语言交换邀约"
```

## 消息生成规则

1. **个性化优先**：优先使用与对方兴趣/背景相关的模板
2. **变量填充**：所有 `{{variable}}` 必须填充实际值
3. **长度控制**：开场白不超过 2-3 句话
4. **文化敏感**：避免过于直接或可能冒犯的表达
5. **真实表达**：鼓励用户用自己的话表达，而非完全依赖模板

## AI 增强

AI 可以基于对方 profile 和用户 profile 生成更个性化的消息：

```
Agent 根据：
- 用户 personality: "内向但真诚，喜欢直接沟通"
- 对方 interests: ["音乐", "旅行"]
- 对话历史: "她问了我为什么学俄语"

生成消息：
"Привет! Ты знаешь, я начал учить русский, потому что хочу лучше понять русскую культуру. 
Особенно музыку — Чайковский просто невероятен! А ты как увлеклась музыкой?"
```

## 模板管理

用户可以：
1. **添加自定义模板**：在 `~/.openvista/social/templates.yaml` 添加
2. **修改现有模板**：直接编辑此文件
3. **标记常用模板**：AI 会学习用户偏好，优先使用常用模板
