# RESOURCES.md Format

`RESOURCES.md` lives at `~/.openvista/learning/RESOURCES.md`. It is the curated set of trusted sources for this language. Knowledge for explainers should be drawn from here, not from parametric guesses. Wisdom comes from the communities listed here.

## Structure

```markdown
# [语言] 学习资源

## Knowledge

- [资源名](URL)
  描述：这个资源涵盖什么，什么时候使用它。

## Wisdom（社区）

- [社区名](URL)
  描述：这个社区的特点，适合什么时候使用。
```

## Rules

- **只选高信任来源。** 首选一手来源、公认专家、同行评审作品、严格审核的社区。如果一个资源是营销伪装成教育，删掉。
- **每个条目都要注释。** 裸链接三个月后就没用了。加一行：它涵盖什么，什么时候用它。
- **按 Knowledge / Wisdom 分组。** 镜像上面的哲学。
- **明确标注缺口。** 如果 mission 需要的领域没有好资源，写 `## Gaps` 部分列出缺失的。这驱动后续搜索。
- **严格修剪。** 一个后来发现是错误的、肤浅的或偏离 mission 的资源应该删除，不要埋藏。五条精来源好过三十条平庸的。
- **记录社区偏好。** 如果用户选择不加入社区，在这里注明，这样后续 sessions 不会反复提议。
