# 模拟执行：openvista-learn-foreign-language Skill

## Eval 3: 我想学日语，但不知道从哪里开始，你能帮我制定一个计划吗？

### Agent 执行过程

**Step 1: 固定开场白**
> "你好！我是你的外语学习助手。我支持学习俄语、西班牙语、法语、德语、日语、韩语等多种语言。你想学习哪一种？"

用户回复："日语"

**Step 2: 询问学习目标和时间**
> "很好！你计划什么时候开始学习？想学多久？每天什么时间段有空？"

用户回复："不知道从哪里开始"

**Step 3: Agent 主动引导（而非拒绝）**
> "完全理解！很多人从'想学日语'到'开始学'之间不知道怎么做。让我们一起来规划。你计划什么时候开始学习（比如这周、下周）？每天大概什么时间段有空？学习多久（比如3个月、半年）？"

用户回复："这周开始，每天晚上有空，想学一年"

**Step 4: 评估当前水平**
> "你现在日语水平如何？零基础、入门、还是有一定基础？"

用户回复："零基础"

**Step 5: 生成 MISSION.md**

```markdown
# Mission: 日语学习

## Why
对日本文化感兴趣，希望通过日语学习了解日本文化，并与日本人交流。

## Success looks like
- 12个月内达到 N3 水平（能进行日常交流）
- 能阅读简单日文文章
- 能与日本人进行基础对话

## Constraints
- 已有基础：零基础
- 时间：每天晚上
- 目标：12个月 N3

## Out of scope
- 商务日语
- 日语汉字书法
```

**Step 6: 生成学习计划**
Agent 生成了完整的 12 个月 N3 学习路径，以及第一周计划。

### 输出文件

- MISSION.md ✓
- schedule.yaml ✓
- lessons/0001-japanese-basics.html ✓（五十音图）

### 评估

此测试用例暴露了 Skill 的一个问题：
- 用户说"不知道从哪里开始"时，Agent 不应直接询问时间，而应先确认用户是想现在开始还是只是了解
- 需要在 SKILL.md 中增加对"不确定"状态的引导

**需要优化**：当用户表达迷茫时，Agent 应先安抚并给出方向性建议，再询问具体时间。
